## 📌 Featured Projects

### 🔧 [octg-quality-agent](https://github.com/dea-cruz/octg-quality-agent)
Statistical quality control agent for OCTG tubular inspection — API 5CT 2-7/8" J55.  
Automated pipeline built with LangGraph that runs descriptive statistics, Cp/Cpk capability analysis, SPC control charts, Shapiro-Wilk normality test, Levene + t-test for lot drift detection, chi-square goodness of fit, and Pearson correlation — with 3 conditional decision points.  
**Tech stack:** Python · LangGraph · scipy · pandas · matplotlib · seaborn · pytest
